# CA-IDS - Context-Aware Intrusion Detection System

## Project Info

Advanced LSTM-based intrusion detection system for vehicular CAN networks with real-time monitoring and threat analysis.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository
```sh
git clone <YOUR_GIT_URL>
```

2. Navigate to the project directory
```sh
cd insightful-app
```

3. Install dependencies
```sh
npm install
```

4. Start the development server
```sh
npm run dev
```
